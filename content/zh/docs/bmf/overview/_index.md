# BMF - 跨平台、多语言、可定制的视频处理框架，具有强大的 GPU 加速能力

**BMF（Babit 多媒体框架）** 是由**字节跳动**开发的跨平台、多语言、可定制的多媒体处理框架。经过 4 年多的测试和改进，BMF 专为熟练应对实际生产环境中的挑战而量身定制。目前广泛应用于字节跳动的视频流、直播转码、云编辑和移动前/后处理场景。该框架每天处理超过 20 亿个视频。

以下是 BMF 的一些主要功能：

**跨平台支持：** 与 Linux、Windows 和 macOS 的原生兼容性，以及针对 x86 和 ARM CPU 的优化。
**易于使用：**BMF 提供 Python、Go 和 C++ API，使开发人员能够灵活地使用他们喜欢的语言进行编码。
**可定制性：** 由于 BMF 解耦架构，开发人员可以通过独立添加自己的模块来增强框架的功能。
**高性能：**BMF 具有强大的调度器，并对异构加速硬件提供强大的支持。此外， **NVIDIA** 一直在与我们合作，为视频转码和 AI 推理开发高度优化的 GPU 管道。
**高效的数据转换：**BMF 提供跨流行框架 （FFmpeg/Numpy/PyTorch/OpenCV/TensorRT） 的无缝数据格式转换、硬件设备之间的转换 （CPU/GPU） 以及色彩空间和像素格式转换。

**BMFLite**是一个客户端跨平台、轻量级、更高效的客户端多媒体处理框架。截至目前，BMFLite 客户端算法已在抖音/西瓜等 App 中应用，服务了超过 10 亿的直播 / 视频播放 / 图片 / 云游戏等场景用户，每天处理视频和图片数万亿次。
